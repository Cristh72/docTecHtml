Introducción

HTML (HyperText Markup Language) y CSS (Cascading Style Sheets) son dos tecnologías fundamentales para la creación de sitios web. HTML se utiliza para estructurar el contenido y definir su significado, mientras que CSS se encarga de dar estilo y presentación a ese contenido. Juntos, forman la base de la mayoría de las páginas web modernas.

HTML: El Esqueleto de la Web

HTML (HyperText Markup Language) es el lenguaje utilizado para estructurar el contenido de una página web. Utiliza etiquetas (tags) para marcar elementos y definir su significado.

Un ejemplo simple de HTML sería la siguiente estructura básica de una página web:

"<html>
<head>
<title>Mi ejemplo</title>
</head>
<body>
<h1>¡Bienvenido!</h1>
<p>Esto es ejemplo.</p>
</body>
</html>"

Explicación de este ejemplo:

  • Las etiquetas <html>, <head>, y <body> definen la estructura de la página.
  • Las etiquetas <h1> y <p> son etiquetas que marcan un encabezado y un párrafo, respectivamente, y agregan significado al contenido.
CSS: Estilizando la Web

CSS (Cascading Style Sheets) es un lenguaje de estilo que se utiliza para controlar la apariencia y el diseño de una página web. Permite definir propiedades como colores, fuentes y márgenes.

Aquí hay un ejemplo de CSS que cambia el color del texto y el fondo de un párrafo:

"p { color: blue; background-color: lightgray;}"

Explicación de este ejemplo:

  • El selector p se utiliza para seleccionar todos los elementos de párrafo en la página.
  • Las propiedades "color" y "background-color" se usan para darles un color al texto y un color de fondo a los elementos p respectivamente.
Separación de Responsabilidades

La separación de responsabilidades en desarrollo web significa mantener HTML y CSS en archivos separados para facilitar la administración y el mantenimiento del código.

En lugar de incluir estilos directamente en el HTML, se utiliza un archivo CSS externo vinculado al documento HTML:

"<link rel="stylesheet" type="text/css" href="estilos.css">"

Explicación:

  • El elemento "link" enlaza nuestra estructura HTML con nuestra hoja de estilos CSS.
  • El archivo estilos.css contiene todas las reglas de estilo, lo que hace que el código sea más organizado y mantenible.
Responsive Design

Responsive Design se refiere a la capacidad de una página web para adaptarse a diferentes tamaños de pantalla y dispositivos. CSS es esencial para lograr esto.

Usando una consulta @media, puedes ajustar el diseño para pantallas pequeñas, como teléfonos móviles:

"@media (max-width: 600px) { body { font-size: 14px; }}"

En este ejemplo, cuando el ancho de la pantalla es igual o menor a 600px, el tamaño de fuente se reduce a 14px para mejorar la legibilidad en dispositivos móviles.

Frameworks y Bibliotecas

Son herramientas que permiten a los desarrolladores web aprovechar soluciones probadas y preestablecidas para crear sitios web de manera más rápida y consistente, sin tener que escribir todo el código desde cero.

Frameworks y bibliotecas CSS como Bootstrap y Foundation proporcionan componentes predefinidos y estilos para agilizar el desarrollo web.

Bootstrap es un framework CSS ampliamente utilizado que ofrece botones, formularios, barras de navegación y otros componentes listos para usar en tu sitio web:

"<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Ejemplo Bootstrap</title>
<link href="https://cdn.jsdelivr.net/npm/bootstrap@4.5.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
<div class="container">
<h1>Ejemplo de Bootstrap</h1>
<p>Este es un botón de Bootstrap:</p>
<button class="btn btn-primary">Haz clic aquí</button>
</div>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@4.5.2/dist/js/bootstrap.min.js"></script>
</body>
</html>"

En este ejemplo:

  • Estamos incluyendo Bootstrap 4 en el encabezado (head) del documento HTML mediante un enlace (link) al archivo CSS de Bootstrap desde un CDN (Content Delivery Network).
  • Hemos creado un botón utilizando las clases de Bootstrap. La clase btn se utiliza para estilizar el botón, y btn-primary establece el color de fondo del botón como azul, que es el estilo de botón primario de Bootstrap.
  • Al final del documento, hemos incluido el archivo JavaScript de Bootstrap desde el mismo CDN. Esto es opcional y se utiliza si planeas utilizar componentes interactivos de Bootstrap que requieren JavaScript.
"Cuando abras este archivo HTML en un navegador, verás un botón estilizado de Bootstrap. Puedes personalizar aún más el aspecto y la funcionalidad de tus elementos utilizando las clases y componentes proporcionados por Bootstrap.".
Conclusión

HTML y CSS son los lenguajes fundamentales para crear páginas web. HTML estructura el contenido, mientras que CSS controla el diseño y la apariencia. La separación de responsabilidades, el diseño receptivo y el uso de frameworks facilitan el desarrollo de sitios web efectivos y atractivos.
